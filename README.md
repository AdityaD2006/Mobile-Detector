# Mobile Country Finder

## Overview
The **Mobile Country Finder** is a web-based application that allows users to determine the country associated with a phone number by using its international dialing code (E.164 format). The tool helps in quickly identifying the origin country of any international phone number.

---

## Motivation
With the increasing use of international phone numbers, it is often necessary to know the country of origin quickly. This project helps users identify the country associated with a phone number without the need for manual reference tables.

---

## Aim
To develop a simple, interactive web application that identifies the country of a phone number using its dialing code.

---

## Objectives
- To allow users to enter a phone number in E.164 format.
- To extract the country code from the number.
- To map the country code to the corresponding country or territory.
- To display the result in a user-friendly manner.

---

## Features
- Supports most international dialing codes.
- Accepts input with or without the `+` symbol.
- Displays country name and the remaining part of the phone number.
- User-friendly interface with responsive design.

---

## Technologies Used
- **Frontend:** HTML, CSS, JavaScript
- **Design:** Responsive UI with Flexbox
- **Logic:** JavaScript mapping for country codes
- **Data:** Static country code mapping (E.164 standard)

---

## How to Use
1. Open `index.html` in any modern web browser.
2. Enter a phone number in E.164 format (e.g., `+91 9876543210`).
3. Click the **Find country** button.
4. The country associated with the dialing code will be displayed.
